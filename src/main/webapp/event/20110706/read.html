应聘必读

职位名称： 互联网杂志阅读体验师
职位薪酬： 年薪十万元（人民币）
职位描述：

  1. 体验麦米网在线杂志阅读功能，反馈界面使用心得，不断提出体验意见和建议，帮助网站持续改进产品，使麦米网成为最优质的在线杂志免费阅读网站；
  2. 这个职位类似产品经理，需要和麦米网用户进行积极的线上互动，持续搜集网友对于网站的意见和建议，并及时反馈至麦米网，便于我们尽快改进技术，使平台的功能日臻完善。


职位福利：

  1. 签订正式劳动合同；
  2. 不限工作地点；
  3. 享受本公司正式员工待遇、享受五险一金。


职位要求：

  1. 应届毕业生，无工作经验者优先。我们需要的是有激情、有创造力、想象力不受限的热力青年；
  2. 有志于投身互联网事业，视野开阔、热爱阅读、喜欢杂志、善于思考的年轻人群；
  3. 对新鲜事物持有极大兴趣，对生活、对人生充满好奇和热望;
  4. 有创意、有激情、有责任感、有企图心的人；
  5. 无学历限制、无相关行业从业经历限制、无户籍地域限制。


职位应聘流程：

  1. 应聘者需要先登录麦米网，注册会员并及时完善相关的个人资料和联系方式。（麦米网将奉送旗下杂志商提供的精美期刊。）
  2. 2011年7月20日-2011年8月20日，应聘者提交麦米网阅读体验报告。在此期间，应聘者必须将自己的体验报上传至麦米网，可以多次提交多份不同的体验报告。体验报告的具体形式不限，可以是图文、视频、漫画等，一切随你喜好。体验•上传•分享，就这么简单！
  3. 2011年8月10日-2011年8月20日，作品投票阶段。在这个阶段，由网友对应聘者的体验作品进行投票，票选前20位的应聘者入围面试。
  4. 2011年8月29日-2011年8月31日，采用国内外先进面试技术，进行面对面选拔面试。（入围者来沪食宿、车费由麦米网承担。）
  5. 2011年9月1日，现场签约仪式。最终的胜出者，将被聘为麦米网的“互联网杂志阅读体验师”，双方签订正式劳动合同。


备注：麦米网“十万元年薪•诚聘互联网杂志阅读体验师” 线下招聘活动，正在全国五大城市热力铺开。从7月初开始，麦米网相继在广州、南京、杭州、上海、北京五大城市举行大型现场招聘会，其他城市没有设立招聘点，如有发现欢迎举报。招聘现场气氛热烈，引来关注无数；职场新鲜人问询者众，简历投递积极踊跃。欢迎各方有志青年前去问询。
